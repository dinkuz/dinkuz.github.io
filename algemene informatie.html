Algemene informatie

Invoer, uitvoer en verwerking van informatie

Invoer en Uitvoer van informatie gaat met behulp van het binair stelsel. Als je informatie invult op je computer zoals een letter dan wordt dit omgezet in een stukje binaire code. Dit stukje binaire code wordt dan weer omgezet naar een letter op je scherm. Informatie wordt verwerkt doordat een computer een combinatie van een binaire code herkent. Deze kan dan vervolgens omgezet worden in van alles.

opslag van data

Data wordt opgeslagen als binaire code, dus het bestaat uit alleen maar 0en en 1en. De computer kan deze dan opslaan op de harde schijf. En als je dan een bestand weer nodig hebt haalt de computer deze data weer van de harde schijf af. Hieronder zie je een plaatje van een hardeschijf.
